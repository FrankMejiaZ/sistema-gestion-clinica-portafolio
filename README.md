# Sistema de Gestión Clínica Integral (Portafolio de Arquitectura)

> **Nota:** Este repositorio funciona como vitrina técnica y documentación de arquitectura. El código fuente se mantiene en un repositorio privado por motivos de confidencialidad y futura comercialización del producto.

## Sobre el Proyecto
Desarrollo *end-to-end* de un sistema web integral diseñado para la gestión operativa, administrativa y médica de una clínica. El proyecto abarca desde la estructuración de la base de datos hasta la creación de interfaces de usuario optimizadas, garantizando la seguridad de los datos médicos y la eficiencia en los flujos de atención.

## Stack Tecnológico

**Frontend:**
* TypeScript
* React (con Vite)
* Tailwind CSS (Diseño UI/UX)
* React Router DOM (Navegación)
* Axios (Consumo de APIs)

**Backend & Base de Datos:**
* TypeScript
* NestJS (Arquitectura escalable)
* Prisma ORM (Gestión y modelado de base de datos)

**Seguridad & Autenticación:**
* JSON Web Tokens (JWT)
* bcrypt (Encriptación de credenciales)
* Gestión de roles y permisos (Médico, Paciente, Administrador)

## Módulos Principales Desarrollados

1. **Admisión y Citas:** Gestión de pacientes, agendamiento y control de flujos de entrada.
2. **Historia Clínica Electrónica (EMR) y Triaje:** Registro centralizado de signos vitales, antecedentes y recetas médicas.
3. **Laboratorio Clínico Avanzado:** Módulo especializado con validación automática de rangos para resultados de exámenes.
4. **Portal del Paciente & Landing Page:** Interfaz pública y segura para la interacción directa con los usuarios finales.
